# Commercial Risk Probability: Markov Inequalities and Convergence Insights

STAT2372 Probability Assignment (2022)

This assignment explored probability inequalities and convergence concepts relevant to commercial risk assessment in finance and insurance. Key tasks included deriving Markov-type bounds, proving convergence in probability/distribution, and verifying results via R simulations.

- Probability theory for risk bounds and stochastic processes  
- Mathematical derivations and proofs  
- R programming for simulation and visualization  
- Application to financial risk evaluation
